# План миграции на pywebview

Миграция Django-приложения на десктопное приложение на базе `pywebview` с Vue 3 (CDN, без сборки), `sqlite3` и `opencv-python-headless`. Целевая платформа — Windows. Распределение Raspberry Pi не требуется.
